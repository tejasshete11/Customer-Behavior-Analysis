# Customer Behavior Analysis Dashboard

## 📌 Overview

The **Customer Behavior Analysis Dashboard** is a Data Analytics project that analyzes customer purchasing behavior to identify trends, revenue patterns, and product category performance. The project combines **Python**, **MySQL**, **SQL**, and **Power BI** to transform raw data into meaningful business insights through an interactive dashboard.

---

## 🎯 Project Objectives

* Analyze customer purchasing behavior.
* Identify revenue and sales trends.
* Compare product category performance.
* Understand customer demographics and subscription behavior.
* Build an interactive dashboard for business decision-making.

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **MySQL**
* **SQL**
* **Power BI**

---

## 🚀 Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing using Python
3. Exploratory Data Analysis (EDA)
4. Data Storage in MySQL
5. SQL Query Analysis
6. Interactive Dashboard Development in Power BI

## 🔍 Key Insights

* Clothing generates the highest revenue and sales.
* Around **73%** of customers are non-subscribers.
* Young adults contribute the highest revenue and sales.
* Customer purchasing behavior varies across product categories and shipping methods.
* Interactive filtering enables deeper analysis across different customer segments.
